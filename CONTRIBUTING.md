# Katkı Rehberi

## Bug Bildirme

1. Önce [Issues](../../issues)'a bak, belki zaten var
2. Yoksa yeni aç: hata mesajı + adımlar + ortam bilgisi

## Özellik Önerisi

Issue aç, ne istediğini kısaca anlat.

## Kod Katkısı

```bash
# Fork & clone
git clone https://github.com/YOUR_USERNAME/sahibinden-mcp-server.git
npm install

# Branch aç
git checkout -b fix/bir-sey

# Commit & push
git commit -m "fix: şunu düzelttim"
git push origin fix/bir-sey
```

PR aç, ne yaptığını yaz.
