# Katkı Rehberi

## Önemli

Bu proje eğitim amaçlıdır. Katkıda bulunarak LICENSE dosyasındaki sorumluluk reddini kabul etmiş olursunuz.

## Hata Bildirimi

1. Önce [Issues](../../issues) sayfasına bak
2. Yoksa yeni aç: hata + adımlar + ortam bilgisi

## Özellik Önerisi

Issue aç, kısaca anlat.

## Kod Katkısı

```bash
git clone https://github.com/YOUR_USERNAME/sahibinden-mcp-server.git
npm install
git checkout -b fix/duzeltme
git commit -m "fix: aciklama"
git push origin fix/duzeltme
```

PR aç.
